---------------------------Part Collection Data------------------------------
object ที่ถูกระบุตำแหน่งทั้งหมด เป็น object ที่ไม่มีการตกขอบของ pixel ทั้งสิ้น
ตำแหน่งของ object ที่ labeled อยู่ในโฟลเดอร์ CSV

ขั้นตอนการเก็บ data จาก Mikulski Archive
1. เอา CSV ไปเข้า STScl แล้วดูว่าตำแหน่งที่กระจุกกันอยู่ตรงไหน
2. จากนั้น เปิดอีก Tab นึงแล้วพิมพ์ว่า M33
3. เลือก Legion จากพื้่นที่้ข้อ 1 แล้ว Save หลังจากนั้นแตกไฟล์
--------------- แปลงภาพ FITS -> jpag------------------------------------------
4. ใช้ fits_to_jpg.ipynb ในการแปลง *** (Flip ภาพให้ตรงกันภาพที่ใช้ใน ds9 ก่อน save โดยใช้คำสั่ง np.flip(data, axis=0))

5. สร้าง bounding box ผ่านไฟล์ data_surveying(M33)

--------------- ทำเฉลยสำหรับ Yolo--------------------------------------------
6. เปิด fits files ในโปรแกรม ds9
* 6.1 หมุนแกนภาพให้จุด x 0 y 0 อยู่ที่ มุมซ้ายบน โดยไปที่ Zoom -> Invert Y
  โดยหลังจากที่ได้ตำแหน่งจาก ds9 ที่จุดเริ่มต้นเป็น 0, 0
  

7. load region จาก ..\regions\global\box_regions.reg
8. save object ที่อยู่ในแต่ละภาพ

###################################################################################
ในขั้นตอน 5 6 7 ทำเพื่อให้ได้ตำแหน่งของ object ที่อยู่ในแต่ละภาพ 
!!! ในขั้นตอนของการเลือก object จะเลือก object ที่สมบูรณ์, ภาพไม่ขาดเท่านั้น !!!
###################################################################################

9. ใช้ ds9_labeling.ipynb ในการสร้างเฉลยจากไฟล์ที่ได้มาจาก ds9

10. แปลงตำแหน่งที่ได้จาก ds9_labeling.ipynb ให้เป็นตำแหน่งรูปแบบ Yolo โดยใช้ position_to_yolo.ipynb

11. ใช้ position_to_YOLO_format.ipynb ในการแปลงจาก pixel position ให้เป็นตำแหน่งเป็น % ของขนาดภาพตามรูปแบบ Yolo

12. Crop รูปภาพ ...

13. Run Model
